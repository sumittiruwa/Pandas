Common Pandas Operations
Reading CSV File
df = pd.read_csv("data.csv")
Display First Rows
print(df.head())
Display Last Rows
print(df.tail())
Dataset Information
print(df.info())
Statistical Summary
print(df.describe())
Selecting a Column
print(df["Name"])
Filtering Data
print(df[df["Age"] > 20])
Adding New Column
df["Salary"] = [20000, 25000, 30000]
Removing Column
df.drop("Salary", axis=1, inplace=True)

Handling Missing Values
Fill Missing Values
df.fillna(0)
Remove Missing Values
df.dropna()
